What is typscript?
-Typescript is build on the top of javascript, which means the code written inside the
-TS will run inside the javascript also.
-browser doesnt understand the TS,so it compiles TS code into JS code at last.
-TS extension is .ts
-With TS we can define the type of the variables, and functions.
-With TS we can write error free code, it will give us error at runtime. So we can easily 
find and fix at the development time itself
